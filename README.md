# BugM2

Firmware and 3D printer data for BugM2.

## Firmware

- `Firmware/BugC2_mecanum`
- `Firmware/General_joy_atoms3_OTA`

## Build

Install PlatformIO, then run:

```sh
cd Firmware/BugC2_mecanum
pio run

cd ../General_joy_atoms3_OTA
pio run
```

## 3D Printer Data

Place STL, STEP, 3MF, or slicer project files in `3D_Printer_Data`.
